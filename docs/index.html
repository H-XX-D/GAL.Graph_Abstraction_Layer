Draft v0.1

GAL Graph Abstraction Layer

A small, inspectable interchange layer for graph-runtime state: nodes, edges, signal wiring, standing operations, and parameter changes.

Specification Roadmap Glossary Dialects Schemas Example

Core Contract

GAL text is designed to round-trip through a canonical AST while preserving graph-runtime meaning. 

GAL text -> AST -> canonical GAL text -> AST

Line Forms

Node

<id> "label" fields...

Edge

<relation>> <target>(weight)

Net

net output op input...

Standing Operation

addf opInstance tick [params...]

Parameter Set

setp target.param value

Dialect Family

Dialects are optional lenses over GAL:netlist, not a mandatory stack. Load only the vocabularies needed for the graph question.

MAL

Memory graphs: claims, evidence, contradiction, and standing memory programs.

DAL

Deployment graphs: services, artifacts, releases, rollout gates, and rollback paths.

PAL

Policy graphs: rules, obligations, approvals, decisions, and exceptions.

AAL

Agent graphs: agents, tools, tasks, plans, handoffs, and receipts.

WAL

Workflow graphs: jobs, steps, gates, triggers, retries, and artifacts.

OAL

Observability graphs: metrics, logs, traces, monitors, alerts, and incidents.

Runtime

HAL, RAL, SAL, EAL, TAL: hardware, resources, state, events, topology, and placement.

Contracts

CAL, IAL: capabilities, grants, interfaces, schemas, and compatibility.

Schema Contracts

JSON Schemas for CLI payloads, registries, runtime state, and verification reports.

Assurance

QAL, FAL, GOVAL, RISKAL, AUDAL, VAL: quality, failure, governance, risk, audit, and verification.

Knowledge

KAL, ReAL, LAL: knowledge, reasoning traces, learning, and calibration.

Minimal Example

@gal netlist.v0
@dialect mal.v0

claim_ab12 "API is degraded" conf(0.82) [kind: claim]
concerns> service_api(0.9)
net alert or2 high_concern stale
addf watch0 tick [topics: api-status] [delta: 0.20]
setp watch0.delta 0.25

Current Tooling

The Python CLI parses, formats, verifies, converts, loads into an in-memory runtime, and publishes schema contracts for adapter-facing JSON payloads.
